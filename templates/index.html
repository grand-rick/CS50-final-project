Welcome to my Cipher web app

My Cipher App is a web application that provides users with the ability to securely communicate in private and secure ways. It is designed to be an easy-to-use tool that affords users with the capability to securely chat and share sensitive information.

It is based on the Caesar cipher model, which encrypts data using a key. However, this is a complex innovation thereof and offers much more security as the key is not only randomly chosen, but also encrypted.

Looking into the future

The app would be a more complex innovation of the Caesar cipher in that it would allow for more than just a simple substitution of letters. The app would instead allow for the substitution of entire words or phrases, making it much more difficult to break the code.
